# GameBot
A bot I made for a game.

# Requirements:

This bot needs a userInfo.json with the following:
`{
  "accounts": [
    {
      "username": "",
      "password": "",
      "task": "Farming Leader",
      "abilities": [
        {
          "key": "f1",
          "castTime": 1,
          "coolDown": 2
        },
        {
          "key": "f2",
          "castTime": 1,
          "coolDown": 2
        }
      ]
    },
    {
      "username": "",
      "password": "",
      "task": "Farming",
      "abilities": [
        {
          "key": "f1",
          "castTime": 1,
          "coolDown": 2
        },
        {
          "key": "f2",
          "castTime": 1,
          "coolDown": 2
        }
      ]
    }
  ]
}`

and a gameAttributes.json with the following:
`{
	"exeName": "",
	"exePath": "",
	"accountPath": ""
}`
