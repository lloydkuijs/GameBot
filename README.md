# GameBot
A bot I made for a game.

# Requirements:

This bot needs a userInfo.json with the following:
`{
  "accounts": [
    {
      "username": "-",
      "password": "-",
      "class": "Farming Leader"
    },
    {
      "username": "-",
      "password": "-",
      "class": "Farming Mage"
    }
  ]
}`
